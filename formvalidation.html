This is my first form validation using html
While you should always validate data on your
servers,there are many benefits to additional
validation of data on the Web page itself.
In many ways, users are annoyed by forms.
By validating form data while the user is filling
it out,the user can know immediately if they've made any mistakes;
this saves the user the time of waiting on an HTTP response,
and saves your server from dealing with bad form input.
This article covers how to go about validating form data
within your form's Web content.Test Test
